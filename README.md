# 微信公众号 Markdown 编辑器

一款专用于微信公众号的 Web 编辑器，可以将 Markdown 转换为公众号文章格式

## 功能

- 解决 `ul` 和 `ol` 元素样式渲染时被微信编辑器重置的问题
- 外链会自动转换为参考文献索引，并且附在文章末尾
- 支持多种字体和样式
